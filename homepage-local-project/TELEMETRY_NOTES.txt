🔑 LA CLEF DE LA SOLUTION: Function() Constructor
════════════════════════════════════════════════════════════════════════════
Problème:
  Webpack analysait statiquement les imports gRPC → erreur "Module not found"
Solution:
  Utiliser Function() constructor pour exécuter le code à runtime:
  const initCode = `
    const { NodeSDK } = require("@opentelemetry/sdk-node");
    // ... autres imports ...
  `;
  new Function(initCode)();  // Webpack ne voit JAMAIS ce code
Résultat:
  ✅ gRPC reste côté serveur
  ✅ Webpack ne le voit jamais au build
  ✅ Build complet en 26.7s
